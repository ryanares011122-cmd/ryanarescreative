Desktop Widget Prototype

硬件监控组件

Hardware Glass

轻量悬浮式桌面监控组件，用于快速查看 CPU、GPU、内存等运行状态。重点是信息层级、透明度控制和日常使用时的误触规避。

A lightweight floating desktop widget for checking CPU, GPU, and memory status, focused on information hierarchy and low-interruption interaction.

设计重点

Design Focus

01

适合置顶查看，不遮挡主要工作区域。

Always-on-top reference without blocking the workspace.

02

信息模块保持窄条化，减少桌面负担。

Compact information bars to reduce visual weight.

03

支持透明度、锁定和鼠标穿透等桌面使用场景。

Supports opacity, lock, and mouse-through usage scenarios.
